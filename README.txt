Require python verion > 3

Sudoku.py consists of the sudoku solver using Naïve Backtracking Algorithm

command to run:

python sudoku.py <input file name>

sudoku_MRV.py consists of the sudoku solver using Smart Backtracking Algorithm with minimum remaining values (MRV)
strategy

command to run:

python sudoku_MRV.py <input file name>
